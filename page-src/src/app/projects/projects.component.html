dynamic-table

Dynamic table component for AngularJS (1.5.x) applications. It allows the addition/removal of columns on runtime without the need of reloading views/collections. This is a simple component that we needed in some of the last projects I worked on, to supply certain functionality. This is just a demo of the working component.
